================================================================================
                              TROUBLESHOOTING
================================================================================

PROBLEM: "ModuleNotFoundError: No module named 'django'"
SOLUTION: In Bash console, run: workon mymovieapi

PROBLEM: "Static files not loading"
SOLUTION: In Bash console, run:
  python manage.py collectstatic --noinput --settings=config.settings.pythonanywhere

PROBLEM: "502 Bad Gateway"
SOLUTION: 
  1. Check error logs in Web tab
  2. Make sure WSGI file is correct
  3. Click Reload button

PROBLEM: "Allowed host mismatch"
SOLUTION: Make sure ALLOWED_HOSTS in pythonanywhere.py includes your domain

PROBLEM: "Database errors"
SOLUTION: In Bash console, run:
  python manage.py migrate --settings=config.settings.pythonanywhere
